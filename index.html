patchwork

secure p2p sharing tool

Download .zip Download .tar.gz View on GitHub

Patchwork is a p2p social-sharing tool for the desktop. Its features include:

  • Public markdown newsfeeds
  • End-to-end encrypted private messaging
  • Communication over WiFi
  • No sign-in, and smooth offline operation
  • GPL licensed

What's interesting about patchwork? It uses the Secure Scuttlebutt protocol.

  1. The network doesnt use hosts or central authorities. Users publish signed data-logs, which are gossiped through the network.
  2. Users are identified by pubkeys. The social profiles and follow-graph are published on the logs to identify users, similar to PGP's Web of Trust.
  3. Log entries are typed JSON entries with content-addressed links to messages, file-blobs, and other logs. Its API is accessible through the Scuttlebot server.
  4. Exchanging only in LAN mode creates a Sneakernet.
