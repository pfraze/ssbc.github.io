patchwork alpha

secure p2p sharing

Patchwork is a p2p social-sharing tool for desktops. Its features include:

  • Public and topical newsfeeds
  • End-to-end encrypted private messaging
  • Internet, WiFi, and offline operation
  • GPL licensed

Closed Alpha: Invite-only

Patchwork is in closed alpha, and is invite-only. Find pfraze, domanic, or mixmix in #scuttlebutt on Freenode to get an invite code.

...or Clone

Cloning with Git makes it easy to update in the future. (Requires node v4.)

git clone https://github.com/ssbc/patchwork.git
cd patchwork
npm install
npm run build:ui
npm start

To update:

cd patchwork
git pull
npm install
npm run build:ui
